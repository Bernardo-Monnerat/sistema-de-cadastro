# 📋 Sistema de Cadastro de Clientes (CRUD)

Um sistema de cadastro via terminal, desenvolvido em Python. O objetivo deste projeto é aplicar e praticar conceitos fundamentais de lógica de programação, como laços de repetição, estruturas condicionais e manipulação de arquivos.

## 🚀 Funcionalidades

O sistema funciona como um CRUD (Create, Read, Update, Delete) completo e possui as seguintes opções:
- **1. Cadastrar cliente:** Adiciona nome, idade, e-mail e telefone.
- **2. Listar clientes:** Mostra todos os cadastros registrados.
- **3. Buscar cliente:** Localiza um cadastro específico digitando o nome.
- **4. Editar cliente:** Permite alterar os dados de um cliente já existente.
- **5. Excluir cliente:** Remove um cadastro do sistema.
- **Persistência de Dados:** Todas as informações são salvas automaticamente em um arquivo `clientes.json`. Isso garante que os dados não sejam perdidos quando o programa é fechado.

## 🛠️ Tecnologias e Conceitos Utilizados

- **Python 3**
- **Biblioteca JSON** (nativa do Python)
- Estruturas de repetição (`while`, `for`)
- Controle de fluxo e condicionais (`if`, `elif`, `else`)
